# FRESH COFFE (FRONT)

## Requerimientos

-   Nodejs (v20.0)
-   React (v18)

## Pasos

Clonar repositorio

```bash
  git clone https://github.com/isaul19/fresh_coffe_front.git
```

Dirigirse al directorio

```bash
  cd ./fresh_coffe_front
```

Instalar dependencias

```bash
  npm install
```

Levantar servidor local

```bash
  npm run dev
```
