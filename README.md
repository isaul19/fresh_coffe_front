# FRESH COFFE (FRONTEND)

## Requerimientos

-   Nodejs (v20.0)
-   React (v18)

## Pasos

Clonar repositorio

```bash
  git clone https://github.com/isaul19/fresh_coffe_frontend.git
```

Dirigirse al directorio

```bash
  cd ./fresh_coffe_frontend
```

Instalar dependencias

```bash
  npm install
```

```bash
  npm run dev
```
